P E R F I C T A

  • A CSG modeller written in the C++ language and accessed through a call interface.
  • Model sheets and wires as well as solids.
  • Define parameterized models within C++, using geometric data types, overloaded operators and geometric transform functions.
  • Create primitives from arbitrary implicit polynomials, trigonometric functions, exponentials and user-defined functions.
  • Define and attach attributes, and exploit inheritance.

 
 


P E R F R A C T A

  • Space division turns CSG into the efficient solid modelling technology.
  • Digital solids are supported en passant
  • Automatic sub-model commoning allows many different versions of a model to be stored efficiently.
  • Use the division strategies provided, or implement your own using the built-in facilities for interval arithmetic.

 
 


Q V A E R E N D O

  • Find interferences, distances and directed distances between objects.
  • Extract facets for graphics.
  • Implement your own tests on divided models.
  • Use programmable Monte Carlo methods together with point-membership tests to obtain integral properties.

 
 


P E R F E C T A

  • Available on Sun, Linux/PC, Microsoft/PC, and Silicon Graphics computers.
  • On-line documentation from the same source as well-known textbooks in geometric computing.
